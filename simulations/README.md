# شبیه‌سازی‌ها — راهنمای بازتولید

سه اسکریپت مستقل، هرکدام یک سؤال پیش‌ثبت‌شده را آزمون می‌کنند.

| اسکریپت | سؤال |
|---|---|
| scenario1_coalition_capture.py | آیا کارتل امنیت-انرژی-اقتصاد غالب می‌ماند؟ |
| scenario2_emergency_institutionalization.py | آیا فیوز ۱۴روزه از نهادینه‌شدن اضطرار جلوگیری می‌کند؟ |
| scenario3_simultaneous_failure.py | آیا پراکندگی جغرافیایی زمان بازیابی را کوتاه می‌کند؟ |

## اجرا
pip install numpy matplotlib
python scenario1_coalition_capture.py

فونت FreeSerif برای نمایش صحیح فارسی در نمودارها لازم است؛ در نبودش اسکریپت باز هم اجرا می‌شود.
